# Pokémon API Search App

A simple and interactive web app that lets users search for Pokémon using the Pokémon API. The app fetches Pokémon data dynamically and displays the result directly in the browser.

## Features

- Search for any Pokémon by name
- Fetch Pokémon data from the Pokémon API using Axios
- Display the Pokémon name and image
- Remove displayed Pokémon cards by clicking on them
- Clean, responsive UI built with Tailwind CSS

## Tech Stack

- HTML
- Tailwind CSS
- JavaScript
- Axios
- Pokémon API

## Project Structure

- `index.html` — Main HTML structure
- `app.js` — Search logic and API integration

## How to Run

1. Open the project folder in your browser.
2. Launch `index.html` directly, or use a simple local server if preferred.
3. Type a Pokémon name in the search box and click **Find**.

## Example

Type a name such as:

- Pikachu
- Charmander
- Eevee

and the app will display the corresponding Pokémon.
